# Mihir Vaidya Portfolio

A modern, interactive portfolio website showcasing my work as a Product Designer with over 2+ years of expertise.

## 🚀 Features

- Modern and minimalist design
- Interactive project cards with custom cursor
- Smooth transitions and animations
- Responsive layout for all devices
- Project showcase with detailed case studies
- Background blur effects and glassmorphism

## 🛠️ Built With

- **React.js** - Frontend framework
- **Tailwind CSS** - Styling and responsive design
- **React Router DOM** - Navigation and routing
- **Custom Animations** - Smooth transitions and hover effects

## 🎨 Design Elements

- Custom cursor with hover effects
- Glassmorphism UI elements
- Dynamic card transitions
- Responsive grid layouts
- Custom fonts:
  - ClashDisplay (Multiple weights)
  - Satoshi (Multiple weights)

## 🏃‍♂️ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/Mihir-Vaidya.git
```

2. Navigate to project directory
```bash
cd Mihir-Vaidya
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```

The application will open in your default browser at `http://localhost:3000`
